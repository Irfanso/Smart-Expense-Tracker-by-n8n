# Smart-Expense-Tracker-by-n8n
AI-powered expense tracking system built using:

* n8n
* Telegram Bot
* Google Sheets
* Ollama AI
* Gmail
* ngrok

## Features

* Expense logging via Telegram
* Automated daily/weekly/monthly reports
* Budget tracking
* Savings goals
* AI financial queries
* Gmail + Telegram notifications

## Tech Stack

* n8n
* Ollama (llama3.1:8b)
* Google Sheets API
* Telegram Bot API
* Gmail API
* ngrok
